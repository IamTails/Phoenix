Multi-Bot Setup


1. Create multiple config.json files in bot settings.
2. It's highly recommended to use a separate API key for each config file (account).
3. "botTimeoutafter" should NOT be the same on all the config files. Keep the bot timeout 5 minutes apart from each other.
4. The "guiPortSocket" on each of the bots MUST NOT be the same. (Example- Make it like 8001,8002,8003...etc for each config file)
5. Fill in your proxy configuration in the config file as follows

"proxyServer" : "ProxyIP",
"proxyPort" : Port,
"proxyType" : "HTTP",
"proxyUsername" : "username",
"proxyPassword" : "password",

6. Once you've done all this, start the bot and headover to hydrabot.info and click Start. If multiple bots appear in the UI, you've successfully set up Multi-bots.
